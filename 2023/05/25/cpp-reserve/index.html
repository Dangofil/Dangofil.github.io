概念总结

封装的概念：将函数放到结构体（类）内部，便于使用结构体（类）内的成员变量

对象的概念：使用结构体（类）创建一个变量，这个变量就叫对象

成员的概念：在结构体（类）内的变量和函数，叫做成员变量，成员函数

父类指针指向子类对象

示例：

这样编写没有任何问题，

因为子类继承了父类，所以子类指针的类型和父类指针指向相同，即都指向x

因此可以用父类指针指向子类的对象，

但是正常情况下只能访问父类的成员，因为子类成员并不属于父类

但这只是编译器的限制，事实上指针就指在那里，完全可以通过pp++来访问子类数据

此时的反汇编

可以发现，通过pp = (Person*)((int*)pp + 2);，就可以访问子类对象，使得level = 3

实际上我们只是将编译器做的事情自己做了而已，通过汇编可以发现，pp->sex = 2和pp = (Person*)((int*)pp + 2);+*(int*)pp = 3;的实现原理几乎一致

所以说，只要对底层内存足够了解，通过指针+强制转型，可以绕过编译器的限制做很多事情
